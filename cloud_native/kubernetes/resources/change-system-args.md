### 修改系统参数

#### 1.修改内核参数
容器和宿主机共享内核，所以创建容内的内核参数就是宿主机的内核参数
  * 修改主机的内核参数，容器内的内核参数也会发生变化
  * 修改容器内的内核参数（需要权限），宿主机的内核参数也会发生变化

#### 2.修改ulimit值
宿主机的ulimit的hard决定了最高上限
* hard值决定了soft值的上限，soft值直接产生影响（即使用时不允许超过soft值）
容器不会继承宿主机设置的ulimit值
* 可以在docker的配置中设置：`daemon.json`
```json
{
  "default-ulimits": {
    "nofile": {
      "Hard": 64000,
      "Name": "nofile",
      "Soft": 64000
    }
  }
}
```
