# data link layer
[toc]

### 概述

#### 1.有两种类型信道

##### （1）点对点信道（PPP协议）
使用一对一的点对点通信方式

##### （2）广播信道（以太网，CSMA/CD协议）
使用一对多的广播通信方式
即发送的数据，连接在线路上的机器都会收到，但只有特定的机器会接受，其他的都会丢弃

#### 2.基本概念

##### （1）link（链路）
一条点到点的物理线路

##### （2）data link（数据链路）
除了物理线路外，还需要通信协议才能进行数据传输
把实现通信协议的硬件和软件加到链路上，就构成了数据链路（常用的方法是网卡）
