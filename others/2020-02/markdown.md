# 基础语法
## 分级标题
```
# xx      //一级标题
## xx     //二级标题
...以此类推
```
## 换行
```
xx        //结尾需要两个空格
```
## 分割线
```
***

```

## 加粗和斜体
```
*xx*        //斜体
**xx**      //加粗
***xx***    //斜体加粗
效果如下：
```
*xx*
**xx**
***xx***
## 上下标
```
<sup>xx</sup>     //上标
<sub>xx</sub>     //下标
效果如下：
```
a<sup>b</sup>
a<sub>b</sub>

# 高级用法
## 列表

```

```

* xx2

### 有序列表
```
1. xx1

```

### 任务列表
```

```
- [ ] xx
- [x] xx
