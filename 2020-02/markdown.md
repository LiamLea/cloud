# 基础语法
## 分级标题
```
# xx      //一级标题
## xx     //二级标题
...以此类推
```
## 换行
```
xx        //结尾需要两个空格
```
## 加粗和斜体
```
*xx*        //斜体
**xx**      //加粗
***xx***    //斜体加粗
效果如下：
```
*xx*
**xx**
***xx***
## 上下标
```
<sup>xx</sup>     //上标
<sub>xx</sub>     //下标
效果如下：
```
a<sup>b</sup>
a<sub>b</sub>
## 插入链接
```
[显示名称](链接到的地址)
效果如下：
```
[test](http://baidu.com)
## 插入图片
```
![xx1](xx2,"xx")

//xx1：如果图片不能显示就显示该文件
//xx2：图片地址
//xx3：将鼠标放图片上显示的文字

效果如下：
```
![无法显示](http://zh.mweb.im/asset/img/set-up-git.gif "图片Title")
